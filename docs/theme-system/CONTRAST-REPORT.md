- Reason: --accent is intentionally NOT inverted between light and dark themes (#fcd34d in both), but --fg-on-accent inverts together with the rest of the text roles (near-black #0c0a09 in light, near-white #e7e5e4 in dark) — in the dark-based themes this puts light text on a background that stayed light.
